### Environment Variables

The following environment variables are required:

- `OPENAI_API_KEY`
- `PRIVATE_INTEGRATION_API_KEY`
- `PRIVATE_INTEGRATION_API_URL`
- `GHL_LOCATION_ID`

### Setup Instructions

The setup instructions for these environment variables are as follows:

1. Add the required environment variables to the `.env.local` file.

```
OPENAI_API_KEY=<your_openai_api_key>
PRIVATE_INTEGRATION_API_KEY=<your_private_integration_api_key>
PRIVATE_INTEGRATION_API_URL=<your_private_integration_api_url>
GHL_LOCATION_ID=<your_ghl_location_id>
```

2. Make sure to replace the placeholders `<your_openai_api_key>`, `<your_private_integration_api_key>`, `<your_private_integration_api_url>`, and `<your_ghl_location_id>` with the actual values for your environment.
